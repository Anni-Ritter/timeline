# Timeline Component Test

This project implements the interactive **Timeline** block according to the Figma design and technical specification.

The component displays several time periods arranged on a circle, smooth transitions between them, and a slider with key events for each period.

---

## How to Run the Project

### 1. Install dependencies

```
npm install
```

### 2. Start the development server
```
npm start
```

The project will be available at:
http://localhost:3000
